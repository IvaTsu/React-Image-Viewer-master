Installation<br>
1.) <code>npm install</code> <br>
2.) <code>npm start</code> <br>
3.) If you want hot reloading feature, skip #3 and type <code>npm run dev</code> instead of <code>npm start</code><br />
4.) <code>http://localhost:8080</code>
